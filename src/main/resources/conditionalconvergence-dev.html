Conditional Convergence

We illustrate the result that if a series is conditionally convergent (i.e., convergent but not absoutely convergent), then there is a subseries that converges to any desired real number (target) for the sequence ((-1)^n)/n.

Specify the target and the desired accuracy to get a partial sum of a subseries that is within the specified error of the target. Note that this partial sum is obtained by including a term if and only if it results in a better approximation of the target, so as the accuracy tends to zero we do get an infinte subseries (i.e., we get partial sums of a fixed subseries).
